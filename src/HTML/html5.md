# html5的特性

1、语义化标签，比如 header、footer、nav等
2、多媒体支持，比如 video、audio
3、Canvas，可以绘制图表或者游戏
4、本地存储，比如 web Storage 和 indexedDB
5、新的表单元素，比如 input 的 type="email", type="range"
6、Web Workers，比如做一些复制的计算逻辑，不阻塞用户界面
7、Web Socket，可以实时聊天
8、地理位置，网页访问用户的地址位置信息
9、支持SVG，可缩放矢量图形
10、离线应用程序，web离线也能工作
11、新事件API，比如 addEventListener
